# Introduction

[Beampipe](https://beampipe.io) is a web analytics tool that aims to be easy, powerful and privacy-conscious.

![Screenshot](/screenshot.png)

Here are a few of the features we offer right out of the box:

- Understand your website's traffic and audience.
- Discover most visited pages, top traffic sources and geographic regions.
- Log custom events like sign-ups, logins or purchases.
- Use goals to better understand your user acquisition funnels.
- **PRO** Integrate with Slack to be notified when specific events occur or receive daily summary reports.
