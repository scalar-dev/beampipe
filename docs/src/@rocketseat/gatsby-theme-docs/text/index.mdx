# Introduction

[Beampipe](https://beampipe.io) is a web analytics tool that aims to be easy, powerful and privacy-preserving.

Here are a few of the features we offer right out of the box:

- Analyse traffic and unique visitor counts over time.
- Discover top visited pages, top traffic sources, top visitor countries etc.
- Set up conversion goals and track users through these stages.
- Log custom events like sign-ups, logins or purchases.
- Configure Slack integration to be notified when specific events occur.
