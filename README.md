## Website

Start developing

```sh
cd web
yarn
yarn dev
```

Run locally at http://localhost:3000

## Docs

Docs website: https://docs.beampipe.io/

```sh
cd docs
yarn
gatsby develop
```

Run locally at
http://localhost:8000
